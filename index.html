HRMS Description

This system was designed to allow organizations to quickly manage their workforce. Users of the HRMS can easily re-arrange and create new elements as part of their work force. Along with this, the HRMS can allow users to quickly and effectively record actions for current personnel.

HRMS Actors

The actors that interact directly with the HRMS are :

  • Guest
  • HR Administrator
  • Manager
  • Employee
  • Applicant

Actors who the HRMS interact with:

  • Account Management System

HRMS Package Structure

The overall composition of the HRMS

Organization Management

The organizational structure can be modified using the HRMS and is available for any guest using the system to view. Here, the HR administrator is able to change the hierarchal structure of the organization and provide new units and positions for these units.

Recruitment Use Cases

Using the HRMS, administrators are able to ppost new positions that are able to be filled. Along with this they can see any applicants that have applied to the system. Any guest can see available positions available as well as submit an application to the organization.

On Board Use Cases

The onboard package allows new hires to input their information for future reference by the ogranization using the hrms system. The employee is also able to change their benefits plan and complete their payroll.

Personnel Management Use Cases

The HRMS allows easy access for its users to modify the records of current employees part of the system. Managers are able to submit reviews for their workforce and employees can view information regarding them. With this information provided by the manager, HR administrators can use the HRMS to determine if the employee should be let go, or promoted. The HRMS also allows the administrator to handle interpersonal relationships as well, such as business expenses, training, and office policy guidelines.

Payroll Use Cases

The HRMS also allows its Administrators to interact with an account management system in which payroll for thee employees could be processed.
